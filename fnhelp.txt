  	 ВВЕДЕНИЕ

    Добро пожаловать в Flex Navigator - программное обеспечение, которое  облегчит 
Вашу работу с данными в формате MS-DOS.


  	 НАЧАЛО РАБОТЫ

    Flex Navigator представляет собой привычную файловую оболочку типа  Norton 
Commander (на PC) или Commander DOS (на Sprinter'е), только выполнен  в 
графическом интерфейсе. Реализованы практически все функции файловой  
оболочки для работы в DOS.
    Если Вы хотите, чтобы Flex Navigator начинал свою работу после включения  
компьютера, то в файле SYSTEM.BAT необходимо прописать полный путь с его  
именем.


  	 ФАЙЛОВАЯ ПАНЕЛЬ

    Файловая панель представляет собой углубление со списком файлов текущей  
директории с полосой прокрутки. Переход от одной панели к другой осуществляется  
при помощи клавиши Tab. Поменять панели местами можно клавишей Ctrl+U.

    Курсорные клавиши - движение по файлам
    PageUp - на страницу файлов назад
    PageDown - на страницу файлов вперед
    Home - в начало списка файлов
    End - в конец списка файлов
    Alt+F1 - сменить носитель для левой панели
    Alt+F2 - сменить носитель для правой панели
    Alt+F3 - просмотр файла встроенным просмотрщиком файлов
    Alt+F4 - редактирование файла встроенным редактором файлов
    Shift+F3 - ввести имя файла для просмотра
    Shift+F4 - ввести имя файла для редактирования
    Ctrl+\ - перечитать корневой каталог
    F1 - помощь
    F3 - просмотреть файл
    F4 - редактировать файл
    F5 - копировать файл(ы)
    F6 - переименовать файл
    F8 - удалить файл(ы)
    F9 - строка меню
    F10 - выход в DOS
    Alt+Enter - информация о файле
    Ctrl+F9 - распечатать файл на принтере
    + - маркировать группу файлов
    - - ремаркировать группу файлов
    * - обратить маркировку
    Ctrl+ - маркировать все файлы
    Ctrl- - ремаркировать все файлы
    Alt+F10 - обновить дерево каталогов
    Ctrl+L - информация о текущем носителе
    Ctrl+F1 - краткий список файлов
    Ctrl+F2 - полный список файлов
    Ctrl+F3 - сортировка по имени
    Ctrl+F4 - сортировка по расширению
    Ctrl+F5 - сортировка по датам
    Ctrl+F6 - сортировка по размеру
    Ctrl+F7 - без сортировки
    Ctrl+F8 - дерево каталогов
    Ctrl+F10 - список всех файлов
    Ctrl+F11 - список запускаемых файлов
    Ctrl+F12 - фильтр для списка файлов
    Alt+A..Z - быстрый поиск файла


  	 СТРОКА DOS

    Строка DOS служит в основном для запуска файлов с параметрами. Для 
активизации  строки необходимо попасть в нее курсором мыши или нажать клавишу 
не относящуюся  к функциональным.

    Ctrl+E - предистория строки
    Ctrl+Enter - поместить в строку текущее имя файла
    Ctrl+[ - поместить в строку полный путь левой панели
    Ctrl+] - поместить в строку полный путь правой панели


  	 МЕНЮ  - File -

    File info... - информация о текущем файле (имя, длина, дата создания,                 
                    атрибут, первый кластер файла)
    Print - распечатать файл на принтере
    Exit - выход в DOS


  	 МЕНЮ  - Select -

    Mark group... - маркировать группу файлов
    Unmark group... - ремаркировать группу файлов
    Mark all - маркировать все файлы
    Unmark all - ремаркировать все файлы
    Invert selection - обратить выделение


  	 МЕНЮ  - Option -

    Refresh tree - обновить дерево каталогов
    Disk info... - информация о текущем носителе
    System information... - информация о системе


  	 МЕНЮ  - View -

    Brief - краткий список файлов
    Full - подробный список файлов
    Tree - дерево каталогов
    All files - список всех файлов
    Execute files - список запускаемых файлов
    *.* - установить фильтр для списка файлов
    Filtr... - фильтр для списка файлов
    Name - сортировать по имени
    Extention - сортировать по расширению
    Data - сортировать по дате
    Size - сортировать по размеру
    Unsorted - без сортировки


  	 МЕНЮ  - Config -

    Setup... - установки для Flex Navigator


  	 МЕНЮ  - About -

    Version... - информация о версии Flex Navigator


  	 ФАЙЛЫ РАСШИРЕНИЙ

    Для подключения внешних программ для работы с файлами существуют файлы  
расширений. Flex Navigator использует три файла расширений:

    fn.ext - главный файл расширений
    fnview.ext - файл расширений прострмотра
    fnedit.ext - файл расширений редактора

    Формат файлов расширений:

    ext:    [drive]:\[path]\filename.ext parametr

        где:   ext - расширение файла
                [drive]:\[path]\filename.ext - внешняя подключаемая программа
                parametr - !.! текущее имя файла
                               !:  текущее устройство
                               !\  текущий путь
